API Basejump: Image Search Abstraction Layer

User stories:

  • I can get the image URLs, alt text, and page URLs for a set of images relating to a given search string.
  • I can paginate through the responses by adding a ?offset=starting-item-nbr parameter to the URL.
  • I can get a list of the most recently submitted search strings by using the path /api/latest/imagesearch

Example image search usage:

https://fcc-image-search.herokuapp.com/api/imagesearch/searchterm

Example pagination:

To get the second page of results: https://fcc-image-search.herokuapp.com/api/imagesearch/searchterm?offset=10
To get the third page of results: https://fcc-image-search.herokuapp.com/api/imagesearch/searchterm?offset=20
