Free Code Camp Assignment

API Basejump: Image Search Abstraction Layer

User stories:

  • I can get the image URLs, alt text, and page URLs for a set of images relating to a given search string.
  • I can paginate through the responses by adding a ?offset=starting-item-nbr parameter to the URL.
  • I can get a list of the most recently submitted search strings by using the path /api/latest/imagesearch

Example image search usage:

https://fcc-img-search.herokuapp.com/api/imagesearch/searchterm

Example image search output (first page of results: items 1-10):

[
  {
    link: image file URL,
    snippet: image description,
    size: image thumbnail URL,
    thumbnailLink: image file URL,
    contextLink: URL for page containing image,
  },
  ...
]

Example pagination:

To get the second page of results (items 11-20):
  https://fcc-img-search.herokuapp.com/api/imagesearch/searchterm?offset=11

To get the third page of results (items 21-30):
  https://fcc-img-search.herokuapp.com/api/imagesearch/searchterm?offset=21


Example list of recent searches:

https://fcc-img-search.herokuapp.com/api/latest/imagesearch

Example search list output:

[
  {
    term: "cats",
    when: "2016-05-23T19:50:59.486Z"
  },
  {
    term: "dogs",
    when: "2016-05-22T14:36:40.827Z"

Example search-by-date usage:

Not part of assignment, added for personal use, not for images, searches not stored.

https://fcc-img-search.herokuapp.com/api/searchbydate/searchterm?days=30?offset=1

Example search-by-date output:

{
  searchInformation: {
    formattedTotalResults: total nbr of results
  },

    {
      title: page title,
      link: page URL,

    },
    ...

}
