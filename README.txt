API Basejump: Image Search Abstraction Layer
============================================

Run "npm install" to install.

Create .env file in root dir with following format (do not commit this file to the repo):
  DB_URL=<mongodb connection string>
  GOOGLE_SEARCH_API_KEY=
  GOOGLE_SEARCH_ENGINE_ID=

Run "npm start" to run. Then go to http://localhost:8080/ in browser and read instructions/usage info there.
